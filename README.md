A simple command-line Rock-Paper-Scissors game written in Python.  
Play against the computer with random choices and see who wins!

---

## Features
- User input with shorthand (`r`, `p`, `s`) or full words (`rock`, `paper`, `scissor`)
- Random computer choice
- Win/Lose/Draw logic
- Clean and simple CLI output

---

## How to Run
1. Clone the repository:
  git clone https://github.com/mussab-alvi/rock-paper-scissors.git
   
2. Navigate into the folder:
  cd rock-paper-scissors

4. Run the game:
  python rps.py

Enter the choice (r/rock, p/paper, s/scissor): r
Your choice is: rock
Computer chose: paper
You lose, Try your luck next time !!!
